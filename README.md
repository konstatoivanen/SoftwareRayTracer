# SoftwareRayTracer
A simple C++ 17 software ray tracer.


## Features
- KD Tree accelerated triangle raycasting.
- Recursive GGX importance sampling.
- Recursive random sampling.

## Config Mode Options
- ggx
- random
- albedo
- normals
- emission

## Third Party
- [tinyobjloader](https://github.com/tinyobjloader/tinyobjloader)
